# 서버 실행 방법

## 방법 1: 일반 실행 (터미널 창 표시)
`start-server.bat` 파일을 더블클릭하세요.
- 서버가 실행되며 터미널 창이 표시됩니다.
- 서버를 중지하려면 터미널 창에서 `Ctrl + C`를 누르세요.

## 방법 2: 백그라운드 실행 (터미널 창 최소화)
`start-server-background.bat` 파일을 더블클릭하세요.
- 서버가 백그라운드에서 실행됩니다.
- 별도 창이 열리지만 최소화할 수 있습니다.
- 서버를 중지하려면 작업 관리자에서 `node.exe` 프로세스를 종료하거나 창을 닫으세요.

## 방법 3: npm 명령어 직접 사용
터미널(CMD)을 열고:
```bash
cd "C:\Users\한\Desktop\테스트\청소 관리앱"
npm run dev
```

## 서버 중지 방법
- 방법 1: 터미널 창에서 `Ctrl + C`
- 방법 2: 작업 관리자에서 `node.exe` 프로세스 종료
- 방법 3: 서버가 실행 중인 창을 닫기
