# 카메라 권한 설정 가이드

## 🚨 현재 문제
카메라 접근에 실패했습니다. 브라우저에서 카메라 권한을 허용해야 합니다.

## ✅ 해결 방법 (브라우저별)

### 📱 모바일 브라우저

#### Chrome (Android)
1. 브라우저 주소창 왼쪽의 **자물쇠 아이콘** 또는 **정보 아이콘** 클릭
2. **"사이트 설정"** 또는 **"권한"** 선택
3. **"카메라"** 항목 찾기
4. **"허용"** 선택
5. 페이지 새로고침

#### Safari (iOS)
1. **설정** 앱 열기
2. **Safari** 선택
3. **"카메라"** 항목 찾기
4. **"사이트별 설정"** 또는 **"권한"** 선택
5. 해당 사이트의 카메라 권한을 **"허용"**으로 변경
6. 페이지 새로고침

#### Samsung Internet (Android)
1. 브라우저 메뉴 (⋮) 클릭
2. **"설정"** 선택
3. **"사이트 권한"** 또는 **"권한"** 선택
4. **"카메라"** 선택
5. 해당 사이트를 찾아 **"허용"** 선택
6. 페이지 새로고침

### 💻 PC 브라우저

#### Chrome
1. 주소창 왼쪽의 **자물쇠 아이콘** 클릭
2. **"사이트 설정"** 클릭
3. **"카메라"** 항목을 **"허용"**으로 변경
4. 페이지 새로고침

또는:
1. Chrome 메뉴 (⋮) → **"설정"**
2. **"개인정보 및 보안"** → **"사이트 설정"**
3. **"카메라"** 선택
4. **"허용"** 목록에 사이트 추가

#### Edge
1. 주소창 왼쪽의 **자물쇠 아이콘** 클릭
2. **"사이트 권한"** 클릭
3. **"카메라"** 항목을 **"허용"**으로 변경
4. 페이지 새로고침

#### Firefox
1. 주소창 왼쪽의 **자물쇠 아이콘** 클릭
2. **"권한"** → **"카메라"** → **"허용"** 선택
3. 페이지 새로고침

## 🔍 추가 확인 사항

### 1. HTTPS 또는 localhost 확인
- 카메라는 **HTTPS** 또는 **localhost**에서만 작동합니다
- 현재 `http://192.168.x.x:3000`로 접속 중이라면:
  - ✅ localhost는 허용됨
  - ⚠️ IP 주소는 브라우저에 따라 제한될 수 있음
  - 💡 **해결책**: PC에서 `http://localhost:3000`으로 접속해보세요

### 2. 카메라 하드웨어 확인
- 다른 앱에서 카메라가 작동하는지 확인
- 카메라가 다른 앱에서 사용 중이 아닌지 확인
- 기기 재시작 시도

### 3. 브라우저 재시작
- 브라우저를 완전히 종료하고 다시 실행
- 권한 설정 후 페이지 새로고침

### 4. 브라우저 업데이트
- 최신 버전의 브라우저 사용
- 오래된 브라우저는 카메라 API를 지원하지 않을 수 있음

## 📝 빠른 해결 방법

### ⚠️ 중요: 사이트별 권한 설정 필요
브라우저의 **일반 카메라 설정**이 아닌 **이 특정 사이트(192.168.x.x:3000)에 대한 권한**을 설정해야 합니다!

### 방법 1: 주소창에서 직접 권한 허용 (가장 빠름)
1. 주소창 왼쪽의 **자물쇠 아이콘** 또는 **경고 아이콘(⚠️)** 클릭
2. **"사이트 설정"** 또는 **"권한"** 선택
3. **"카메라"** 항목 찾기
4. **"허용"** 또는 **"항상 허용"** 선택
5. 설정 화면을 닫고 페이지 새로고침

### 방법 2: Chrome 설정에서 사이트별 권한 설정
1. Chrome 메뉴 (⋮) → **"설정"**
2. **"개인정보 및 보안"** → **"사이트 설정"**
3. **"카메라"** 선택
4. **"허용"** 목록에 `192.168.x.x:3000` 추가
   - 또는 **"차단"** 목록에서 제거
5. 페이지 새로고침

### 방법 2: 브라우저 설정에서 전체 허용 (개발용)
⚠️ **주의**: 개발/테스트 환경에서만 사용하세요.

**Chrome:**
1. `chrome://settings/content/camera` 접속
2. **"사이트에서 카메라 사용 요청 전에 확인"** 체크 해제
3. 또는 **"허용"** 목록에 사이트 추가

## 🆘 여전히 문제가 있나요?

### ⚠️ 가장 흔한 원인
**브라우저의 일반 카메라 설정이 아닌, 이 특정 사이트에 대한 권한을 설정해야 합니다!**

### 체크리스트
- [ ] **주소창의 자물쇠 아이콘에서 이 사이트의 카메라 권한이 "허용"인가?**
- [ ] 브라우저의 일반 설정이 아닌 **사이트별 권한**을 확인했는가?
- [ ] 다른 앱에서 카메라가 작동하는가?
- [ ] 브라우저를 완전히 종료하고 다시 실행했는가?
- [ ] 페이지를 새로고침했는가?
- [ ] `localhost:3000`으로 접속해보았는가? (IP 주소 대신)

### 추가 해결 방법
1. **시크릿/프라이빗 모드에서 테스트**
   - 시크릿 모드에서 접속하여 권한 다시 요청

2. **브라우저 캐시 삭제**
   - 설정 → 개인정보 → 캐시 삭제

3. **다른 브라우저로 테스트**
   - Chrome, Safari, Edge 등 다른 브라우저 시도

4. **기기 재시작**
   - 모바일 기기 또는 PC 재시작

## 💡 개발자 참고사항

### 카메라 API 요구사항
- **HTTPS 필수** (localhost 제외)
- **사용자 권한 필요**
- **getUserMedia API 지원 브라우저**

### 지원 브라우저
- ✅ Chrome 53+
- ✅ Firefox 36+
- ✅ Safari 11+
- ✅ Edge 12+
- ✅ Samsung Internet 5.0+

### 에러 코드별 해결
- **NotAllowedError**: 권한 거부 → 브라우저 설정에서 허용
- **NotFoundError**: 카메라 없음 → 하드웨어 확인
- **NotReadableError**: 카메라 사용 중 → 다른 앱 종료
- **OverconstrainedError**: 요구사항 불일치 → 카메라 설정 확인
